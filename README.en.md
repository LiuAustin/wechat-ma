# 微信第三方开放平台代小程序实现业务

#### Description
假如说，有多个业务，功能模式相同的公众号/小程序，如果只是小程序开发，那是不是需要复制多套代码，改appid信息，在微信公众号后台，配置域名服务器以及密钥等繁琐的信息，每改一个提交发布一次，进行重复的步骤。随着要维护的公众号/小程序数量逐步增加，需要投入的成本也随之增加。有没有想过，只需要开发一套公众号/小程序代码，以之为模板，把在微信公众号后台做的那些事都搬到我们自己的系统中。
